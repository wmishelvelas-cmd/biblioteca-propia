# ⚡ Inicio Rápido

## 🚀 Forma más rápida (Sin requisitos)

1. **Descarga los archivos** o clona el repositorio
2. **Abre `index.html`** en tu navegador
3. ¡**A jugar!** 🎮

Eso es todo. No necesitas nada más.

---

## 📦 Con servidor Node.js (Opcional)

### Requisitos:
- [Node.js](https://nodejs.org/) instalado

### Pasos:

```bash
# 1. Ve a la carpeta del juego
cd ruta/a/subway-surfers

# 2. Instala dependencias
npm install

# 3. Inicia el servidor
npm start

# 4. Abre tu navegador en
http://localhost:3000
```

---

## 🌐 GitHub Pages (Hosting gratuito)

1. Sube todos los archivos a GitHub
2. Ve a **Settings** del repositorio
3. Busca **Pages** en el menú lateral
4. Selecciona **main** como rama
5. Tu juego estará en: `https://tu-usuario.github.io/subway-surfers`

---

## 🎮 Controles

- **Flechas izquierda/derecha**: Cambiar carril
- **Flecha arriba**: Saltar
- **Flecha abajo**: Deslizarse
- **Móvil**: Desliza el dedo en cualquier dirección
